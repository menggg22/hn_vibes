# HN Vibes

A weekly log of what Hacker News is feeling — themes, mood, standouts, and sparks for builders.

Not a dashboard. Not a data pipeline. Just a running editorial record of what the HN crowd is excited about, angry at, or obsessing over each week.

## What's here

`vibes/YYYY/week-WW-mmmDD.md` — one file per week, appended each time I run `/hn`.

Each entry:
- **Top themes** — what dominated the front page
- **Vibe** — the mood in 1-2 sentences
- **Standout** — one story that felt genuinely new or important
- **Spark** — something that connects to building, shipping, or ideas worth following

## Why

HN is a signal about what technically-minded builders care about. Reading it weekly, and writing down what you notice, compounds. After a few months you start to see what keeps coming back, what's peaking, what's fading.

Useful for timing when to post your own work. See [JanBussieck/hn-skill](https://github.com/JanBussieck/hn-skill) for a more rigorous pre-post scoring tool.
