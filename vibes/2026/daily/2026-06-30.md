## 🔭 Depth — ml-infra / on-device ML / agent-systems

*What's the trend and what sparks curiosity.*

**ml-infra & on-device ML**: The Qwen 3.6 27B story reflects a consolidating consensus that ~27B quantized models are the local-dev sweet spot — fast enough, capable enough, and private enough. The Apple Neural Engine architecture post surfacing alongside it suggests the hardware side of the equation is getting serious attention.
- [Apple Neural Engine: Architecture, Programming, and Performance](https://news.ycombinator.com/item?id=48702825) — 94pts — rare low-level ANE programming coverage; worth reading if you're building anything that needs sustained on-device inference on Apple silicon

**agent-systems**: Herdr, Fleet, XSDR, and Ornith-1.0 all shipped or surfaced today, suggesting the agent tooling layer is hitting a cambrian explosion moment — everyone is building slightly different takes on orchestration, multiplexing, and observability. The Micro-Agent post (beat frontier models with collaboration inside model API) adds a capability claim worth stress-testing.
- [Herdr: Agent multiplexer that lives in your terminal](https://news.ycombinator.com/item?id=48714802) — 143pts — terminal-native agent multiplexer is the Unix-philosophy answer to the GUI agent dashboards proliferating everywhere; the 90c discussion is worth mining for real usage patterns

**Deep dives**:
- [What happens when you run a CUDA kernel?](https://news.ycombinator.com/item?id=48718863) — 194pts — with everyone running local inference, understanding the kernel dispatch path is newly practical rather than purely academic
- [WATaBoy: JIT-Ing Game Boy Instructions to WASM Beats a Native Interpreter](https://news.ycombinator.com/item?id=48720190) — 169pts — JIT-to-WASM as an emulation strategy has implications beyond Game Boy; the performance delta over native interpretation is the interesting number to examine
