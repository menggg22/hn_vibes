## 🔭 Depth — ml-infra / on-device ML / agent-systems

*What's the trend and what sparks curiosity.*

**ml-infra & on-device ML**: The DeepSeek v4 + GPT-5.5 same-day drop is stress-testing the community's ability to benchmark and contextualize rapidly; meanwhile the IBM Quantum / /dev/urandom story (329pts) quietly asks whether current quantum backends are even worth the latency. The Lambda Calculus Benchmark for AI (128pts) suggests builders want evaluation frameworks that can't be gamed by training data.
- [Replace IBM Quantum back end with /dev/urandom](https://news.ycombinator.com/item?id=47897647) — 329pts — if random noise matches quantum output quality on real workloads, the infrastructure cost calculus for near-term quantum is worth revisiting today
- [Lambda Calculus Benchmark for AI](https://news.ycombinator.com/item?id=47900506) — 128pts — symbolic reasoning benchmark that's hard to memorize; worth reading alongside GPT-5.5 and DeepSeek v4 coverage to check their actual reasoning quality

**agent-systems**: Claude 4.7 ignoring stop hooks (89pts, 84c in Ask/Tell) is the sharpest signal — the community is discovering that agent control primitives are not as reliable as the docs imply. The Karpathy-wiki Show HN and the Agent MCP Studio both assume agents need persistent, structured memory; the question is whether that memory layer becomes commodity infrastructure or a moat.
- [Tell HN: Claude 4.7 is ignoring stop hooks](https://news.ycombinator.com/item?id=47895029) — 89pts — concrete failure mode for production agent pipelines; worth reading before deploying any hook-dependent safety logic
- [Show HN: A Karpathy-style LLM wiki your agents maintain](https://news.ycombinator.com/item?id=47899844) — 221pts — Git as agent memory backend is elegant and auditable; the architecture question (who writes, who reads, conflict resolution) is the interesting part

**Deep dives**:
- [There Will Be a Scientific Theory of Deep Learning](https://news.ycombinator.com/item?id=47893779) — 338pts 150c — rare HN thread where the comments are likely as good as the piece; worth reading for the minority view that current empiricism will eventually yield to principled theory
