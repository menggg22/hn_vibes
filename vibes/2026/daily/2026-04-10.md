## 🔭 Depth — ml-infra / on-device ML / agent-systems

*What's the trend and what sparks curiosity.*

**ml-infra & on-device ML**: The story today isn't new model releases — it's the infrastructure layer fracturing. Developers are routing around Claude Code's pricing toward OpenRouter, questioning CDN reliability (BunnyCDN silently losing files), and scrutinizing every plugin that touches their prompts. Trust in hosted ML infra is at a low.
- [Reallocating $100/Month Claude Code Spend to Zed and OpenRouter](https://news.ycombinator.com/item?id=47700972) — 297pts — worth reading to understand how cost-sensitive the agentic coding workflow actually is and what the switching cost looks like in practice

**agent-systems**: The Vercel/Claude Code prompt-reading story and the research-driven agent post together sketch a tension: agents need broad context access to be useful, but that same access is exactly what makes them a privacy and trust liability. The field hasn't resolved this.
- [The Vercel plugin on Claude Code wants to read your prompts](https://news.ycombinator.com/item?id=47704881) — 255pts — the specific permission model being asked for and why it's different from normal plugin behavior is the thing to understand here
- [Research-Driven Agents: When an agent reads before it codes](https://news.ycombinator.com/item?id=47706141) — 138pts — a constructive counterpoint; argues pre-task research phases improve output quality, which reframes the "context access" debate

**Deep dives**:
- [How NASA built Artemis II's fault-tolerant computer](https://news.ycombinator.com/item?id=47704804) — the design constraints for life-critical redundancy are a useful foil against "move fast" AI deployment assumptions
- [Reverse engineering Gemini's SynthID detection](https://news.ycombinator.com/item?id=47709130) — watermarking as a trust mechanism is only as good as its resistance to exactly this kind of analysis; read to calibrate how robust these systems actually are
- [BunnyCDN has been silently losing our production files for 15 months](https://news.ycombinator.com/item?id=47710845) — a slow-failure postmortem that reveals how long silent data loss can go undetected in CDN infrastructure; directly relevant to anyone running media-heavy production systems
