## 🔭 Depth — ml-infra / on-device ML / agent-systems

*What's the trend and what sparks curiosity.*

**ml-infra & on-device ML**: The model-release conveyor belt (DeepSeek v4, GPT-5.5, GPT-5.5 Pro in API all on the same day) is creating infrastructure whiplash — teams haven't finished evaluating the last model before the next drops, and the research thread on number representations across models hints at convergence patterns that could inform architecture decisions.
- [Different Language Models Learn Similar Number Representations](https://news.ycombinator.com/item?id=47890873) — 89pts — if models independently converge on the same internal representations, it has deep implications for model merging, distillation, and alignment transfer

**agent-systems**: The Claude stop-hook failure and the broader quality degradation complaints are exposing a fragility in agent pipelines that assumed model behavior would be stable — builders are now shipping monitoring and canary tooling (CC-Canary, HNswered, Agent Vault) as a direct response.
- [Tell HN: Claude 4.7 is ignoring stop hooks](https://news.ycombinator.com/item?id=47895029) — 66pts — a seemingly minor API behavior change that breaks entire agent control loops; the comments will be a field guide to agentic failure modes

**Deep dives**:
- [There Will Be a Scientific Theory of Deep Learning](https://news.ycombinator.com/item?id=47893779) — 100pts — worth reading today specifically because two frontier models just dropped with minimal mechanistic explanation; the gap between capability and theory has never felt wider
- [Spinel: Ruby AOT Native Compiler](https://news.ycombinator.com/item?id=47887334) — 299pts — AOT compilation for a dynamic language is technically ambitious; in a week of vibe-coded Python wrappers, this is serious systems work
- [Bitwarden CLI compromised in ongoing Checkmarx supply chain campaign](https://news.ycombinator.com/item?id=47876043) — 847pts — supply chain attack on a security tool is exactly as bad as it sounds; if you use Bitwarden CLI in any automated pipeline, read this before anything else today
