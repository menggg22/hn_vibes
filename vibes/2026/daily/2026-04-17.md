## 🔭 Depth — ml-infra / on-device ML / agent-systems

*What's the trend and what sparks curiosity.*

**ml-infra & on-device ML**: The on-device story is quietly strengthening — Darkbloom using idle Macs for private inference and Qwen3.6 running locally on a laptop (and beating Opus 4.7 at pelicans) both point toward a world where cloud inference is optional, not mandatory. The privacy angle is becoming the wedge.
- [Darkbloom – Private inference on idle Macs](https://news.ycombinator.com/item?id=47788542) — 468pts — distributed private inference across idle Apple Silicon; worth watching whether the "idle Mac" framing unlocks a consumer network effect.

**agent-systems**: Agents are moving from demos to infrastructure — Cloudflare shipping a dedicated agent inference layer, Android CLI for 3x faster app builds via agents, and Codex hacking a Samsung TV all suggest the agentic surface area is expanding faster than security thinking can follow.
- [Cloudflare's AI Platform: an inference layer designed for agents](https://news.ycombinator.com/item?id=47792538) — 223pts — edge-native agent inference; if Cloudflare's network becomes the default agent runtime, that's a significant architectural lock-in moment.
- [Codex Hacked a Samsung TV](https://news.ycombinator.com/item?id=47791212) — 201pts — agentic system autonomously exploiting consumer hardware; the security implications land differently now that agents are mainstream.

**Deep dives**:
- [AI cybersecurity is not proof of work](https://news.ycombinator.com/item?id=47791236) — 192pts — argues that AI security tooling doesn't actually raise the cost floor for attackers; essential counterpoint to the Codex TV hack hype.
- [€54k spike in 13h from unrestricted Firebase browser key accessing Gemini APIs](https://news.ycombinator.com/item?id=47791871) — 375pts — concrete cost-explosion case study; read alongside the Durable Object $34k alarm loop story for a pattern about how cloud billing has no circuit breakers.
- [US v. Heppner (S.D.N.Y. 2026) no attorney-client privilege for AI chats](https://news.ycombinator.com/item?id=47778920) — 178pts — legal precedent that AI conversation logs aren't privileged; quietly enormous for anyone using AI in professional/legal contexts.
