## 🔭 Depth — ml-infra / on-device ML / agent-systems

*What's the trend and what sparks curiosity.*

**ml-infra & on-device ML**: The story today is inference cost collapse — Xiaomi dropping API prices 99% and Eagle 3.1's speculative decoding collaboration between three teams signals the infra layer is commoditizing fast. On-device momentum continues with the Rosalind genomics toolkit running whole-genome pipelines on a laptop, showing Rust as the serious contender for local ML pipelines.
- [Rosalind: A genomics toolkit in Rust running whole-genome pipelines on a laptop](https://news.ycombinator.com/item?id=48222600) — 114pts — worth watching as a proof-of-concept that "frontier-class" bio workloads can escape the cloud entirely with the right language choice.

**agent-systems**: The "sleep-like consolidation" paper for LLMs is the most intellectually interesting agent story today — the hypothesis that LLMs need an offline consolidation phase (analogous to sleep) to generalize better has direct implications for how long-running agents should be architected. Meanwhile the Show HN cross-agent filesystem messaging and the skills-for-humanity structured reasoning prompt library both suggest practitioners are building scaffolding that frontier labs haven't shipped yet.
- [A sleep-like consolidation mechanism for LLMs](https://news.ycombinator.com/item?id=48281226) — 179pts — if the mechanism holds, it reframes training schedules and fine-tuning as sleep cycles, not just gradient steps; read this before the Twitter takes arrive.

**Deep dives**:
- [Using AI to write better code more slowly](https://news.ycombinator.com/item?id=48272984) — 1135pts 417c — the most important honest-practitioner AI post in weeks; read the comments for the real signal on where AI coding tools are actually failing.
