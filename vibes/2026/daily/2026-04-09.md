## Personal Signal

*ml-infra / on-device ML / agent-systems — what's the trend and what sparks curiosity.*

**ml-infra & on-device ML**:
The on-device fine-tuning story is maturing fast — Apple Silicon is now a serious training target, not just inference. The MegaTrain claim about 100B+ full-precision training on a single GPU is the headline stress-test: if real, it collapses the assumption that large-model training requires clusters.
- [MegaTrain: Full Precision Training of 100B+ Parameter LLMs on a Single GPU](https://news.ycombinator.com/item?id=47689174) — 264pts — the architectural trick enabling this (likely memory tiling / offload strategy) is worth understanding regardless of whether the benchmark holds
- [Gemma 4 Multimodal Fine-Tuner for Apple Silicon](https://news.ycombinator.com/item?id=47680309) — 222pts — multimodal fine-tuning on-device is the next frontier; this is an early practical implementation worth benchmarking

**agent-systems**:
Two distinct agent patterns surfaced today: tooling that wraps existing interfaces for agents (TUI-use, Skrun, Safari MCP) and orchestration scaffolding that gives agents isolated environments (disposable servers for Claude Code). The first wave is about surface-area expansion; the second is about reliability and parallelism — both are active frontiers.
- [TUI-use: Let AI agents control interactive terminal programs](https://news.ycombinator.com/item?id=47692661) — 37pts — making legacy TUI programs agent-accessible without API wrappers is a real unsolved integration problem
- [Skrun – Deploy any agent skill as an API](https://news.ycombinator.com/item?id=47689319) — 44pts — the skill-as-microservice pattern is trying to solve agent composability; worth watching whether it converges with MCP
- [Claude Managed Agents](https://news.ycombinator.com/item?id=47693047) — 145pts — Anthropic's own managed agent architecture; read alongside the OpenClaw ban to understand the platform's intended vs. actual developer story
- [Muse Spark: Scaling towards personal superintelligence](https://news.ycombinator.com/item?id=47692043) — 278pts, 296c — high comment engagement suggests the "personal superintelligence" framing is both attracting and irritating people; the debate is the signal
