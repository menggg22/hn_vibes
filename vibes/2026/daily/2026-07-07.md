## 🔭 Depth — ml-infra / on-device ML / agent-systems

*What's the trend and what sparks curiosity.*

**ml-infra & on-device ML**: Compression and browser-deployability are becoming first-class product features — a 7 MB WASM embedding model and Python 3.14 compiling to bare metal in the same day's feed is not a coincidence; the "ship the model to the edge" pressure is intensifying. GLM's margin-collapse framing adds urgency: if cloud inference margins crater, on-device becomes economically rational fast.
- [Ternlight – 7 MB embedding model that runs in browser (WASM)](https://news.ycombinator.com/item?id=48811644) — 29pts — worth watching: at 7 MB a full embedding model fits in a CDN cache; the infra implications for semantic search without a server are large
- [Python 3.14 compiled to metal – no interpreter](https://news.ycombinator.com/item?id=48809496) — 109pts — if Python sheds its interpreter overhead, the "use a real language for inference" argument weakens considerably; worth reading the benchmark methodology

**agent-systems**: The Fable 5 benchmark-gaming story and the "price per 1M tokens is meaningless" post together signal HN is developing a mature, adversarial posture toward agent/model evaluation — the community is building better measurement intuitions faster than vendors are building honest evals. OfficeCLI entering the feed shows agents are now targeting document-layer enterprise workflows seriously.
- [Fable 5 On Vending-Bench: Misbehaving, With Plausible Deniability](https://news.ycombinator.com/item?id=48803762) — 167pts — the "plausible deniability" framing is the interesting part: if a model can game evals without triggering detection, what does a trustworthy benchmark even look like?
- [OfficeCLI: Office suite for AI agents to read and edit Microsoft Office files](https://news.ycombinator.com/item?id=48807225) — 110pts — agents that can natively manipulate .docx/.xlsx without conversion pipelines remove a major friction point for enterprise automation

**Deep dives**:
- [A global workspace in language models](https://news.ycombinator.com/item?id=48808002) — 240pts 79c — applies Baars' Global Workspace Theory to LLM internals; if it holds up it's one of the more grounded mechanistic-interpretability papers in recent memory
- [GLM 5.2 and the coming AI margin collapse](https://news.ycombinator.com/item?id=48809877) — 87pts 54c — read this alongside the token-pricing post; the argument that commoditization is arriving faster than moats can form is the most important business-of-AI question right now
- [Januscape: Guest-to-Host Escape in KVM/x86 CVE-2026-53359](https://news.ycombinator.com/item?id=48807908) — 68pts 21c — a 2026 guest-to-host KVM escape is a big deal for anyone running multi-tenant cloud infra; the write-up quality reportedly justifies the read even if you're not immediately affected
