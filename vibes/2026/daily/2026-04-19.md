- [Measuring Claude 4.7's tokenizer costs](https://news.ycombinator.com/item?id=47807006) — 684pts — the interesting spark: if tokenizer choice is now a cost lever, what does that mean for model-agnostic infra layers that abstract over it?

- [Libretto – Making AI browser automations deterministic](https://news.ycombinator.com/item?id=47780971) — 130pts — worth watching: if you can make browser agents deterministic, the same techniques likely generalize to any stateful environment agent
