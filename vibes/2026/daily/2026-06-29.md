## 🔭 Depth — ml-infra / on-device ML / agent-systems

*What's the trend and what sparks curiosity.*

**ml-infra & on-device ML**: Inference cost and routing are the dominant problems right now — three separate projects this week attack the question of "which model, where, at what cost" rather than model quality itself, suggesting the community has accepted that good-enough models are abundant and the infrastructure layer is the real moat.
- [Wayfinder Router: deterministic routing of queries between local and hosted LLM](https://news.ycombinator.com/item?id=48704373) — 108pts — deterministic (not probabilistic) routing is an interesting design choice worth interrogating; what does "deterministic" actually mean when the query content varies?
- [AMD Strix Halo RDMA Cluster Setup Guide](https://news.ycombinator.com/item?id=48703258) — 222pts — Strix Halo's unified memory makes multi-node consumer GPU clusters newly viable; this guide is the community figuring out the playbook in real time

**agent-systems**: The agent conversation has matured past "can it work" into "what goes wrong at scale" — the OpenAI Codex sensitive-files issue and the MRI second-opinion story both expose the same problem: agents operate with access and authority that users haven't fully modeled, and the failure modes are serious.
- [A way to exclude sensitive files issue still open for OpenAI Codex](https://news.ycombinator.com/item?id=48706714) — 172pts — an unresolved GitHub issue becoming an HN story signals community frustration reaching boiling point; agentic coding tools with filesystem access need explicit sandboxing contracts
- [Knowledge Distillation of Black-Box Large Language Models](https://news.ycombinator.com/item?id=48712420) — 22pts — distilling closed models via API outputs skirts licensing grey zones; worth reading as the legal landscape around model weights tightens

**Deep dives**:
- [Historical memory prices 1960–2026](https://news.ycombinator.com/item?id=48710092) — 128pts — with HBM constraints shaping every AI infrastructure decision in 2026, grounding yourself in the 66-year cost curve reframes whether current prices are a floor or a blip
- [DLL that was not present in memory despite not being formally unloaded](https://news.ycombinator.com/item?id=48705910) — 129pts — the kind of deep Windows loader archaeology that only surfaces when someone has a genuinely baffling production bug; useful for anyone building agents that shell out to native code
- [Tokenmaxxing is dead, long live tokenmaxxing](https://news.ycombinator.com/item?id=48708795) — 105pts 122c — the high comment-to-score ratio means the thesis is contested; worth reading the thread to understand where prompt-efficiency orthodoxy is cracking
