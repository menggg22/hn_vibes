## 🔭 Depth — ml-infra / on-device ML / agent-systems

*What's the trend and what sparks curiosity.*

**ml-infra & on-device ML**: Efficiency is the dominant story — Orthrus-Qwen3 getting 7.8× tokens/forward pass and δ-mem targeting online memory compression both point to a community that's moved past "bigger model" and into "same output, less compute." The Kioxia/Dell 10PB-in-2RU story is the hardware analog: density as the new frontier.

- [Orthrus-Qwen3: up to 7.8×tokens/forward on Qwen3, identical output distribution](https://news.ycombinator.com/item?id=48154865) — 217pts — "identical output distribution" is the key claim; if it holds under scrutiny this is a significant inference cost reduction with immediate practical value
- [δ-mem: Efficient Online Memory for Large Language Models](https://news.ycombinator.com/item?id=48158506) — 189pts — online (streaming) memory compression rather than static KV cache management; worth reading alongside the agent memory Show HN to see what's missing in current stacks
- [Kioxia and Dell cram 10 PB into slim 2RU server](https://news.ycombinator.com/item?id=48161997) — 105pts — training dataset storage economics are changing when this much NAND fits in 2U; implications for who can afford to run serious ML infrastructure

**agent-systems**: The mood is critical rather than celebratory — "AI psychosis" dominates, the CTF post mourns capability spillover, and the Claude Design lock-in thread shows that trust in AI-first products is fragile. Meanwhile, the engineering side continues quietly: Zerostack's Rust-native coding agent and Statewright's visual state machines show builders still investing in the infrastructure layer.

- [Zerostack – A Unix-inspired coding agent written in pure Rust](https://news.ycombinator.com/item?id=48164287) — 75pts — "Unix-inspired" framing is doing work here: composability and pipes as an antidote to monolithic agent frameworks; architectural philosophy worth examining

- [I tried to make Claude make me money on open-source bounties](https://news.ycombinator.com/item?id=48164229) — 30pts — low score but high signal: a real-world economic test of coding agent capability with an honest results report

**Deep dives**:
- [A 0-click exploit chain for the Pixel 10](https://news.ycombinator.com/item?id=48148460) — 428pts 227c — Pixel 10 is current hardware; a 0-click chain means no user interaction required, making this immediately relevant to anyone with an Android device in their threat model
