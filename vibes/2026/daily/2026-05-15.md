## 🔭 Depth — ml-infra / on-device ML / agent-systems

*What's the trend and what sparks curiosity.*

**ml-infra & on-device ML**: The on-device story is bifurcating — one branch is distillation (Needle's 26M tool-calling model), the other is gaming/inference benchmarking on consumer silicon (RTX 5090 vs M4 MacBook Air). The implicit question both branches are answering: what's the minimum viable hardware for serious inference?
- [RTX 5090 and M4 MacBook Air: Can It Game?](https://news.ycombinator.com/item?id=48137145) — 454pts — framed as gaming but really a proxy benchmark for local inference horsepower on two very different memory architectures; worth reading for the unified memory vs. VRAM discussion

**agent-systems**: Anthropic is flooding the zone — Claude for Small Business, Claude for Legal, Codex on mobile, and a Claude Code skill-building guide all landed within 48 hours. HN's reaction is split between "this is the platform moment" and "I just lost all my projects after unsubscribing." Reliability and lock-in are the emerging fault lines.
- [A Claude Code and Codex Skill for Deliberate Skill Development](https://news.ycombinator.com/item?id=48130679) — 216pts — meta: a guide for using AI coding agents to *learn* rather than just execute; interesting counterweight to "AI is making me dumb"

**Deep dives**:
- [First public macOS kernel memory corruption exploit on Apple M5](https://news.ycombinator.com/item?id=48139219) — first public M5 kernel exploit is a milestone; read for the memory tagging / PAC bypass technique details which will matter for the whole ARM ecosystem
- [New arXiv policy: 1-year ban for hallucinated references](https://news.ycombinator.com/item?id=48140922) — arXiv drawing a hard institutional line is a bellwether for how academic infrastructure will police AI-generated content; the enforcement mechanism is what to study
- [What's in a GGUF, besides the weights – and what's still missing?](https://news.ycombinator.com/item?id=48138332) — if you're shipping local models, GGUF's metadata gaps are a practical problem right now; this is the reference post for understanding what the format does and doesn't carry
