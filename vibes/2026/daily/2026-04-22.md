## 🔭 Depth — ml-infra / on-device ML / agent-systems

*What's the trend and what sparks curiosity.*

**ml-infra & on-device ML**: Quantization philosophy is getting a formal treatment with the Ternary Bonsai post arguing top-tier intelligence at 1.58 bits, while on-device 3D generation on Apple Silicon signals that the local inference curve is steeper than the cloud narrative admits. The KV cache summarization paper landing on the front page suggests researchers are publishing in public now, not waiting for conferences.

- [Ternary Bonsai: Top Intelligence at 1.58 Bits](https://news.ycombinator.com/item?id=47812749) — 222pts — if ternary weights hold up at scale this rewrites the economics of edge deployment; worth reading the methodology skeptically
- [High-Fidelity KV Cache Summarization Using Entropy and Low-Rank Reconstruction](https://news.ycombinator.com/item?id=47823549) — 59pts — entropy-guided cache compression could dramatically extend effective context windows without architectural changes

**agent-systems**: Two distinct signals today — builders pivoting *away* from building agents toward maintaining them (Daemons), and HN's editorial voice pushing back on anthropomorphic framing ("Less human AI agents, please"). The CrabTrap LLM-as-judge proxy is the most technically interesting new entry: using an LLM to police another LLM in production is either brilliant or a recursion trap.

- [CrabTrap: An LLM-as-a-judge HTTP proxy to secure agents in production](https://news.ycombinator.com/item?id=47850212) — 53pts — runtime behavioral guardrails via proxy is a pattern that could become standard infra; the Rust name is a nice touch
- [Less human AI agents, please](https://news.ycombinator.com/item?id=47845429) — 128pts — the HN crowd is actively theorizing about what agent UX should look like post-hype; comment thread is a design critique session worth mining

**Deep dives**:
- [The Vercel breach: OAuth attack exposes risk in platform environment variables](https://news.ycombinator.com/item?id=47851634) — OAuth-chained platform attacks are underreported; if you store secrets in CI/CD env vars this is required reading today
